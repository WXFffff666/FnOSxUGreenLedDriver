# Roadmap

## v1.0 — 手动控制（已完成）

- [x] 手动开关 6 个 LED（电源、网络、磁盘1-4）
- [x] 全部开启 / 全部关闭
- [x] 拟物化 UI（DXP4800 金属外壳风格）
- [x] fnpack 标准项目结构，一键构建
- [x] ugreen_leds_cli 静态编译打包，无外部依赖

## v1.1 — 状态同步

将 LED 显示与 NAS 真实状态绑定，不再只是手动开关。

- [ ] **磁盘状态检测**：轮询 `/proc/mdstat` 或 `lsblk`，LED 反映磁盘是否在线、读写活动
- [ ] **网络状态检测**：读取 `/sys/class/net/` 接口状态，LED 反映网络是否连通、有数据传输
- [ ] **电源状态**：已开机即显示，支持系统事件（关机关闭所有 LED）
- [ ] **LED 模式切换**：每个 LED 支持「手动 / 自动」模式，自动模式下跟随硬件状态
- [ ] **状态轮询服务**：后台定时采集状态，前端 WebSocket 或 polling 实时更新

## v1.2 — 多设备支持

抽象硬件层，支持更多 UGREEN NAS 型号。

- [ ] **设备探针**：安装时自动检测型号（读取 DMI、设备树或 PCI 设备列表）
- [ ] **LED 映射配置**：不同型号的 LED 数量和 I2C 地址不同，改为配置文件驱动
- [ ] **已知型号适配**：

| 型号 | 盘位数 | LED 数 | I2C 地址 |
|------|--------|--------|----------|
| DXP4800 | 4 | 6 | 0x3a |
| DXP4800 Plus | 4 | 6 | 待验证 |
| DXP6800 Pro | 6 | 8 | 待验证 |
| DXP8800 Pro | 8 | 10 | 待验证 |
| DXP2800 | 2 | 4 | 待验证 |

- [ ] **用户自定义**：wizard 中支持手动配置 LED 数量与 I2C 参数，兼容未适配型号

## v1.3 — 高级功能

- [ ] **定时策略**：夜间自动调暗/关闭 LED，白天恢复正常
- [ ] **事件通知**：磁盘故障、网络断开时 LED 闪烁告警
- [ ] **亮度调节**：如果硬件支持 PWM，允许调节 LED 亮度
- [ ] **多语言**：英文 UI 支持

## v2.0 — 生态整合

- [ ] **fnOS 通知中心**：磁盘异常时推送系统通知
- [ ] **快捷面板**：桌面小组件显示 LED 状态概览
- [ ] **开发者文档**：LED 驱动适配指南，方便社区贡献新设备
