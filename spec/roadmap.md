# Roadmap

## v1.0.x — 基础能力（已完成）

- [x] 手动开关 LED（电源、网络、磁盘）
- [x] 全部开启 / 全部关闭
- [x] 拟物化 UI（DXP4800 金属外壳风格）
- [x] fnpack 标准项目结构，一键构建
- [x] ugreen_leds_cli 静态编译打包，无外部依赖
- [x] 非标端口 19580，避免 8080 冲突
- [x] LED 状态持久化（`var/led_state.json`），启动时自动恢复
- [x] 多设备自动探测（`ugreen_leds_cli all -status` 探测可用 LED 和当前硬件状态）
- [x] DMI 型号辅助识别（DX/DXP 已知型号系列）
- [x] ATA 盘位映射，包含 DXP6800 上游特殊顺序
- [x] 动态 UI：根据探测/配置的盘位数显示对应磁盘格
- [x] 盘位配置 API（`POST /api/config`），支持手动切换 2/4/6/8 盘位模式
- [x] 重置配置（`POST /api/reset`），清空本地配置并回到初始化页面

## v1.1 — 状态同步

将 LED 显示与 NAS 真实状态绑定，不再只是手动开关。

- [x] **磁盘活动检测**：轮询 `/sys/block/*/stat`，自动模式下用闪烁反映读写活动
- [x] **网络活动检测**：读取 `/sys/class/net/` 接口状态与流量计数，自动模式下反映数据传输
- [ ] **电源状态**：已开机即显示，支持系统事件
- [x] **LED 模式切换**：每个 LED 支持「关闭 / 常亮 / 自动」模式
- [x] **状态轮询服务**：后台定时采集状态，前端轮询 `/api/status` 实时更新

## v1.3 — 高级功能

- [ ] **定时策略**：夜间自动调暗/关闭 LED，白天恢复正常
- [ ] **事件通知**：磁盘故障、网络断开时 LED 闪烁告警
- [ ] **亮度调节**：利用 CLI 的 `-brightness` 参数
- [ ] **颜色配置**：利用 CLI 的 `-color` 参数
- [ ] **多语言**：英文 UI 支持

- [ ] **fnOS 通知中心**：磁盘异常时推送系统通知
